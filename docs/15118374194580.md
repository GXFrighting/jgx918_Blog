# <span id = "5">6. 为什么拍照模糊</span> 
## 1. 请保持镜头玻璃片清洁
## 2. 移动相机对准不同距离的物品时，相机会重新进行对焦。建议移动相机后稳定1-2秒时间给相机对焦，再按下快门。
